# Personal Website Project

This project is a personal website that showcases self-introduction, research interests, and current publications. It is structured to provide a clear and organized presentation of the content.

## Project Structure

```
personal-website
├── src
│   ├── index.html       # Main HTML file containing the website content
│   ├── css
│   │   └── style.css    # Styles for the website
│   ├── js
│   │   └── main.js      # JavaScript for interactive features
│   └── assets           # Directory for images and other assets
├── package.json         # npm configuration file
└── README.md            # Project documentation
```

## Features

- **Self-Introduction**: A section that provides information about the individual, including background and experience.
- **Research Interests**: A dedicated area to outline specific research topics and areas of expertise.
- **Current Publications**: A list of recent publications, including links and descriptions.

## Usage

1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Open `src/index.html` in a web browser to view the website.
4. Modify the content in `src/index.html`, `src/css/style.css`, and `src/js/main.js` as needed to personalize the website.

## Installation

To install any necessary dependencies, run:

```
npm install
```
